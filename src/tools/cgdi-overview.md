# CGDI MB Overview

This tool is designed to **program keys and read Mercedes modules**, especially helpful when:

- All keys are lost
- You need to read EIS data and calculate passwords
- You want to write or erase keys via infrared or OBD :contentReference[oaicite:4]{index=4}

## Key Features

- Adds keys to Mercedes‑Benz cars even when all original keys are lost
- Fastest key collection ~1:50 and password calculation ~40s with a high success rate
- Works with both BE keys and original Mercedes keys
- Can read EIS data via OBD or infrared
- Displays EIS model, key status and other metadata :contentReference[oaicite:5]{index=5}

## Basic Workflow

### 1. Connect to the Vehicle

- Plug the CGDI MB tool into the **OBD port** of the car
- Ensure the car’s battery has sufficient charge

### 2. Read EIS Data

- Use the software to read the EIS unit data
- This gives you the module info needed for programming

### 3. Calculate Password

- The tool will automatically calculate the password needed to program keys
- This is often done in under a minute :contentReference[oaicite:6]{index=6}

### 4. Add or Program New Keys

- Connect the key adapter and follow on‑screen instructions
- Insert or tap the key when prompted

### 5. Verify Keys

- Use the software to confirm the new key works with the vehicle
