## Basic Workflow

### 1. Connect to the Vehicle

- Plug the CGDI MB tool into the **OBD port** of the car
- Ensure the car’s battery has sufficient charge

### 2. Read EIS Data

- Use the software to read the EIS unit data
- This gives you the module info needed for programming

### 3. Calculate Password

- The tool will automatically calculate the password needed to program keys
- This is often done in under a minute :contentReference[oaicite:6]{index=6}

### 4. Add or Program New Keys

- Connect the key adapter and follow on‑screen instructions
- Insert or tap the key when prompted

### 5. Verify Keys

- Use the software to confirm the new key works with the vehicle

# CGDI PROG MB Mercedes-Benz Key Programmer

## Product Description

**CGDI PROG MB Mercedes-Benz Key Programmer** is a professional device designed for **key programming, module reading/writing, password calculation, and diagnostics** on Mercedes-Benz vehicles.

It supports a wide range of Mercedes models, including advanced **keyless systems**, and can:

- Create new keys from scratch
- Clone existing keys
- Read and write **EIS / EZS**
- Read, write, and repair **ELV / ESL**
- Perform password calculation
- Program **ECU / TCU / ISM**

CGDI MB is a versatile and reliable tool for technicians working with Mercedes-Benz electronic systems.
